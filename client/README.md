# Client

Frontend client built with Vite and React.

## Scripts

- `npm run dev` starts the development server.
- `npm run build` creates a production build.
- `npm run preview` previews the production build locally.
- `npm run lint` runs ESLint.

## Installed Packages

- `react-router-dom`
- `axios`
- `framer-motion`
- `recharts`
- `lucide-react`
